# QR Analyzer
This is a simple QR Code Generator and Decoder.
<br>Runs on Windows Only

STATUS: On Development

## Features
- Easy to Use
- Generate QR Codes
- Read QR Codes
- One-Click COPY
- Customized UI
> CONS: This can't Decode some customized QR Codes
